Heading #

Headings are used as the titles of each major section of a page in the interface.

Usage #

Use for the title of each top-level page section. The default element of heading are h2

Basic Usage #

preview

Any title should here 

<template>
  <Heading>
    Any title should here
  </Heading>
</template>
