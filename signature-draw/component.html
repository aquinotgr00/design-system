Color #

Set pen color with prop color, default is #000000

preview
<template>
  <p-signature-draw color="#73185E" />
</template>

Binding v-model #

Result of drawing image is store in v-model value. It's encoded in base64-dataURI format.

preview

result

<template>
  <p-signature-draw v-model="result" />
</template>

<script setup>
const result = ref('')
</script>
